The Power of Universal Gates: NAND & NOR

June 30, 2026 • 7 min read

Did you know that you don't actually need AND, OR, and NOT gates to build a computer? In fact, many real-world processors and memory chips are built using almost entirely NAND gates. Why? Because NAND and NOR gates are Universal Gates.

What is a Universal Gate?

A universal gate is a logic gate that can be used to construct all other logic functions. This means you can build an AND gate out of NAND gates, an OR gate out of NAND gates, and a NOT gate out of NAND gates. This is incredibly efficient for hardware manufacturing, as it allows silicon fabricators to stamp out billions of identical transistors.

The NAND Gate (NOT-AND)

A NAND gate produces a 0 only if both inputs are 1. In all other cases (if either input is 0), it produces a 1.

Building a NOT Gate from a NAND Gate

How do we turn a NAND gate into an inverter (NOT gate)? We simply tie both of its inputs together! When both inputs are fed the same signal `A`, the NAND gate acts exactly like a NOT gate. Try the interactive demo below to see this in action.

Notice how tying the inputs together inverses the signal.

Building an AND Gate from NAND Gates

To build an AND gate, you simply take a NAND gate and invert its output. Since we just learned how to build an inverter out of a NAND gate, you just connect the output of the first NAND gate into both inputs of a second NAND gate.
