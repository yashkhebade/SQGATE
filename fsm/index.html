How to use FSM Designer

  • Select: Click states/transitions to select them. Drag on canvas to pan. Scroll to zoom.
  • Add State: Select the tool, then click anywhere on the canvas to place a new state node.
  • Add Transition: Select the tool, then drag from one state to another.
  • Edit Properties: Use the right sidebar to define module name, inputs, outputs, and state encoding.
  • Simulate: Open the "Simulate" tab to set input values and clock the FSM to see state changes.

About SQGATE

      
1. Visual Circuit Simulator: Interactively build and simulate complex digital logic circuits using a drag-and-drop interface.

      
2. FSM Designer: Graphically draw state diagrams and automatically generate synthesizable Verilog HDL code.

      
3. Logic Quest Game: Master Boolean concepts by building specific truth tables using a restricted inventory of logic gates.

      
4. K-Map Solver: Simplify Boolean algebra visually with support for up to 6 variable Karnaugh maps.
