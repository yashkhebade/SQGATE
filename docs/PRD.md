# Product Requirement Document (PRD)

## 1. Product Overview
**SQGATE** is a free, privacy-first, client-side digital logic circuit simulator tailored for computer science and electrical engineering students. It provides a web-based, zero-latency environment for designing, simulating, and debugging digital architectures.

## 2. Target Audience
- **Primary:** Engineering and computer science students (e.g., BTech ECE/CS) learning digital logic.
- **Secondary:** Hobbyists and educators looking for an accessible, no-install logic simulator.

## 3. Core Features
- **Visual Circuit Simulator:** Drag-and-drop logic gates, wiring, real-time signal propagation.
- **FSM Designer:** Finite State Machine design with direct Verilog/VHDL export.
- **K-Map Solver:** Interactive Karnaugh Map solver supporting up to 6 variables.
- **Logic Quests/Puzzles:** Gamified learning environment to master gates.
- **Offline & PWA:** Installable as a Progressive Web App for offline usage.
- **Pro Version:** Paid tier (handled via Lemon Squeezy) unlocking premium features, templates, and removing watermarks.

## 4. Monetization Strategy
- **Freemium Model:** Core tools are free.
- **SQGATE Pro:** Premium tier available via a one-time purchase or subscription, providing advanced exports, exclusive components, and ad-free/watermark-free experiences.
